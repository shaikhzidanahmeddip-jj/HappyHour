How to use:
- Download MelonLoader from https://github.com/LavaGang/MelonLoader.Installer/
- Install v0.7 or higher to Liar's Bar. It should auto find the directory.
- Downlaod the latest release `HappyHour.dll` from the releases page.
- Put the `HappyHour.dll` in the `Mods` folder within your Liar's Bar Game Files.
